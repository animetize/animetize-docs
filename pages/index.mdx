# Introduction

Welcome to Animetize! we build APIs, websites and stuff related to anime or manga. 

# Why?
 
We are here to build an API that working just like Consumet. We are doing this because, Consumet's `api.consumet.org` is currently not working because of way too many people using it.
So in order to solve the problem, we build Animetize which can be used as we can make a whole bunch of domains in `vercel`.

## What is Animetize-API?

An API to effortlessly access information on **anime** or **manga** to empower your applications with **streamlined data retrieval**.
